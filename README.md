Numerama Spider
==================

An example web crawler project for Scrapy.

Requirements
--------------
- python 2.7
- Scrapy 1.0.5
- pyquery 1.2.11

```
    pip install -r requirements.txt
```

Usage
-----
```
    scrapy crawl numerama -o items.json -t json
```

Examples
--------
This url: http://www.numerama.com/ could yield an article dictionary
